# Portal de Professores

Um projeto simples em React para listar professores e suas disciplinas, incluindo um formulário de contato básico. Feito em formato Single Page Application (SPA).

## Stack
- React + TypeScript
- Vite
- Tailwind CSS v4

## O que foi feito
- Navegação em página única entre o Início e os perfis de 3 professores.
- Cada perfil exibe as disciplinas lecionadas (com curso e semestre).
- Formulário de contato funcional na interface que, ao enviar, exibe um modal com os dados preenchidos.
